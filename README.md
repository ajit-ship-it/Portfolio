# AutoPay & Account Funding — UX Discovery Case Study

A single-page case study site documenting the discovery work I led for a new
**AutoPay** and **Account Funding** capability on a credit-union messaging and
payments platform.

The work synthesized four client stakeholder conversations into:

- Four shared user-pain themes
- Four stakeholder personas (compliance / growth / operations / collections)
- A consolidated P0 → P2 priority list the product team could ship against

> Client names are anonymized as A / B / C / D for this public write-up.

## View

Open `index.html` directly, or run a tiny local server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy on GitHub Pages

1. Push to GitHub
2. Settings → Pages → Source: `Deploy from a branch` → `main` / `/ (root)`
3. Wait ~30 seconds, then open the Pages URL
